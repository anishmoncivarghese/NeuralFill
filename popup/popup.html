Setup Steps
1Go to chrome://flags/#prompt-api-for-gemini-nano → set to Enabled
2Go to chrome://flags/#optimization-guide-on-device-model → set to Enabled BypassPerfRequirement
3Click Relaunch at the bottom of the flags page
4Wait 15–30 minutes — Chrome downloads the ~1.7 GB model silently in the background
5If model still unavailable, free up at least 22 GB disk space and relaunch Chrome again
⚠ Disk space is the #1 reason Gemini Nano won't load.
Chrome requires ~22 GB free to download and run the model. Even if flags are correct, the model silently stays unavailable if your disk is full.
